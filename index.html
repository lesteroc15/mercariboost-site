MercariBoost

Bulk relist your Mercari listings in one click. Free Chrome extension.

Mercari's algorithm rewards fresh listings. Top sellers relist every 20–40 days. With 100+ listings that's hours of clicking. MercariBoost does it in minutes.

Free. No signup. No payments. Local-only data.

What it does

  • Bulk relist — select multiple listings, click once, watch them all relist
  • Stale-listing dashboard — see which items need a refresh, color-coded by age
  • Listing manager — search, sort, and filter your Mercari closet
  • Progress overlay — watch each relist step happen; no guessing if it's running
  • Recovery safety net — if anything fails mid-relist, the listing is preserved with one-click retry
  • Reliable — daily selector monitoring; fixes shipped within 24h of any Mercari UI change

Privacy-first

  • Your Mercari login never leaves your browser
  • All listing data is stored locally in Chrome
  • No tracking, no telemetry, no ads
